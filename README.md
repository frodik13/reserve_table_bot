# Telegram-бот для бронирования теннисного стола

Бот позволяет бронировать теннисный стол на 15-минутные слоты, отслеживать активные игры и рассылать уведомления всем подписчикам.

## Возможности

- **Бронирование** — выбор слота из доступных (с текущего времени, с шагом 15 мин), ввод имени, подтверждение
- **Уведомления** — все подписчики получают сообщение при новом бронировании
- **Начало игры** — кнопка "Начать игру" с вводом имени, рассылка уведомлений
- **Конец игры** — кнопка видна только тому, кто начал игру; рассылка об окончании
- **Расписание** — просмотр броней на сегодня

## Требования

- Python 3.11+
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))

## Установка

```bash
# Клонировать репозиторий
git clone <repo-url>
cd reserve_table

# Создать виртуальное окружение
python -m venv .venv
source .venv/bin/activate      # Linux/macOS
# .venv\Scripts\activate       # Windows

# Установить зависимости
pip install -r requirements.txt

# Настроить переменные окружения
cp .env.example .env
# Открыть .env и вставить токен бота
```

## Настройка `.env`

```env
TELEGRAM_BOT_TOKEN=your_token_here
DB_PATH=reserve_table.db
TZ=Europe/Moscow
```

| Переменная | Описание | По умолчанию |
|---|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен от @BotFather | **обязательно** |
| `DB_PATH` | Путь к файлу SQLite | `reserve_table.db` |
| `TZ` | Часовой пояс для отображения времени | `UTC` |

## Запуск

```bash
python bot.py
```

Бот запускается в режиме long polling. Для production-развёртывания рассмотрите webhook через nginx/caddy.

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Подписаться на уведомления и открыть меню |
| `/book` | Забронировать стол |
| `/schedule` | Расписание на сегодня |
| `/cancel` | Отменить текущее действие |

## Как это работает

1. Пользователь отправляет `/start` → подписывается на уведомления
2. Нажимает "Забронировать стол" → выбирает время из доступных слотов → вводит имя → подтверждает
3. Все подписчики получают уведомление о брони
4. Нажимает "Начать игру" → вводит имя → все уведомлены → появляется кнопка "Закончить игру"
5. Только инициатор может нажать "Закончить игру" → все уведомлены
